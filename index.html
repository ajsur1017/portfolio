My Works



FitnessDOOD




FitnessDOOD was an application that I built as my capstone project for General Assembly. Hosted on Netlify, it is a CRUD (Create Read Update Delete) application that was built using Ruby on Rails with PostgreSQL for the backend database which is hosted on Heroku. The frontend uses Vue.js with vue-router and milligram.css. This application was designed to help me keep track of what I eat/drink during the day as well as some activity. If you are anything like me you struggle to keep track of all of those little things with everything else going on in our crazy lives.


Currently this application does not have any Authentication, however I currently have a completed backend with JWT/bcrypt authentication that just require some front end work to be completed. Coming soon!

Events-N-Stuff




Events-N-Stuff is a application that was built as a part of a small team of developers where I was tasked with implementing JSON web token authentication as well as construction the backend database using express with MongoDB. The end result was a CRUD application that you can store, organize and even mark you attendance for different events. Features a CSS Carousel and uses Bootstrap.css for several design components. Some future goals the team and I have is to use this frontend application with a third part event API that stores and updates events nationwide!



PokéSearch




PokéSearch is a small application built for mobile that uses an Ajax function to make a call to a thrid party API by searching using the Pokémon name or ID number. The Next button generates a random number to use as a search parameter to display a random Pokémon to the user. Some unuique things here was the use of a default image rather than leaving the display area blank.



Itinerary Planner




This Itinerary day planner is an application built with Ruby on Rails using PostgreSQL for the backend database hosted on Heroku. The frontend which is live on Netlify, was buily using ReactJS. A CRUD application that is simple enough to understand even for my elderly grandparents, help you stay on track with you busy schedule throughout the day. Coming soon to this application in a future update is the ability to have a unique profile to organize your own personal day.
